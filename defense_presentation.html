📊 Project Overview

Research Problem

The COVID-19 pandemic generated unprecedented volumes of scientific literature. Researchers need efficient tools to collect, analyze, and assess bibliometric data quality.

Solution

A comprehensive Streamlit web application integrating multiple academic APIs with quality assessment and report generation.

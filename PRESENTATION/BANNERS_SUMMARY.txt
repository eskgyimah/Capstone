🖨️ PRINTING RECOMMENDATIONS
================================================================================

FOR VERSION 2 POSTER (48" x 60"):

Print Specifications:
• Resolution: 300 DPI (best) or 150 DPI (good)
• File Format: PNG or PDF
• Color Mode: RGB
• Paper Type: Matte or semi-gloss poster paper
• Finish: Professional poster finish

Where to Print:
1. Campus Print Services
   - Most affordable
   - Usually $20-40
   - Check with UCC library or IT services

2. Local Print Shops
   - FedEx Office, Staples, etc.
   - $30-60 per poster
   - Same-day or next-day service

3. Online Services
   - Vistaprint, Printful, Poster Burner
   - $40-80 (includes shipping)
   - 3-7 days delivery

4. Professional Print Houses
   - Highest quality
   - $60-100+
   - Best for important presentations

Preparation Checklist:
☐ Export to PNG at 300 DPI
☐ Verify file size (should be large, 50-200 MB)
☐ Check colors on screen (may differ slightly when printed)
☐ Save backup on USB drive
☐ Email file to print shop OR upload online
☐ Request matte or semi-gloss finish
☐ Specify: 48 inches wide × 60 inches tall
☐ Allow 1-3 days for printing

✏️ CUSTOMIZATION OPTIONS
================================================================================

Easy Edits (No Coding Required):
1. Open HTML in text editor (Notepad, VS Code)
2. Find text between <p> or <h1> tags
3. Change text as needed
4. Save file
5. Refresh browser to see changes

Colors:
• Find: background: linear-gradient(135deg, #006633 0%, #004d26 100%);
• Change hex codes: #006633 = UCC Green
• Alternative greens: #008844, #00AA55, #006B3F

Statistics:
• Find stat-number elements
• Update numbers: 472, 281, 97.52
• Update labels below numbers

Advanced Customization:
• Fonts: Change font-family in CSS
• Layout: Adjust padding, margins
• Sizes: Modify width, height values
• Effects: Adjust shadow, blur values

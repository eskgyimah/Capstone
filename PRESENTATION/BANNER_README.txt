1. Open presentation_banner.html in browser
2. Press: Cmd + Shift + 4

4. File automatically saved to Desktop

1. Open presentation_banner.html in Chrome or Edge
2. Press F12 (opens Developer Tools)
3. Press Ctrl + Shift + P (opens Command Palette)
4. Type: "screenshot"
5. Select: "Capture node screenshot"
6. Click on the banner element
7. PNG automatically downloads

FIREFOX:
1. Open presentation_banner.html in Firefox

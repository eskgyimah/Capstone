
Windows quickstart:

1) Open PowerShell
2) Activate your venv if any
3) Install deps:
   pip install streamlit pandas numpy plotly matplotlib
4) Run production app:
   streamlit run "C:\Users\eskgy\Desktop\CAPSTONE\CHATGPT\production_ucc_system_new.py"
5) Or run standalone orchestrator:
   streamlit run "C:\Users\eskgy\Desktop\CAPSTONE\CHATGPT\ucc_orchestrator_app_new.py"
